CouncilCare

IT Repair Request Management System — a simple way for organizations to log, track, and resolve IT issues.

Log in
What CouncilCare does
  • Users submit IT repair requests with details and attachments.
  • Technicians claim requests and update progress (In Progress / Resolved).
  • Admins manage users, assign work, and monitor activity.
Who it’s for

Built for teams that want to replace paper/WhatsApp-style reporting with a clean, trackable process.

Regular Users IT Technicians Administrators
Technology
Backend: Django
Frontend: Django Templates + Bootstrap
Auth: Django authentication + role-based access
Database: SQLite (dev) / PostgreSQL (production)
Uploads: File attachments supported
Version Control: Git + GitHub
Important note about the database
This live demo should use PostgreSQL for reliable data storage. If you use SQLite on cloud hosting, data resets after redeploys/restarts because the file storage is temporary.
